# PumpFun Bumper Bot

A full-featured Telegram bot for Solana/ETH/BSC services with multi-wallet generation, real payment verification, token info lookup, and Discord webhook logging.

## Features
- Multi-chain wallet generation (SOL, ETH, BSC) per user
- Real on-chain payment verification
- Real-time token info from DexScreener + Jupiter
- Admin panel: ban/unban, chat relay, broadcast
- Discord webhook for all events
- GitHub push for user/payment records
